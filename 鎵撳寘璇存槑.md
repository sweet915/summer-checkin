# 暑假自律打卡 - App打包说明

## 方案一：PWA（推荐，最简单）

无需打包，直接部署即可。用户可以将网页添加到手机主屏幕，体验接近原生App。

### 部署步骤

1. 将 `app-mobile` 文件夹上传到任意Web服务器（如GitHub Pages、Vercel、Netlify、腾讯云COS等）
2. 用户用Safari/Chrome打开网页地址
3. 点击"分享"→"添加到主屏幕"
4. 即可像App一样使用，支持离线缓存

### 文件清单
- `index.html` — 主程序
- `manifest.json` — PWA配置
- `service-worker.js` — 离线缓存
- `icon-192.png` — 图标
- `icon-512.png` — 大图标

## 方案二：Cordova打包（原生App）

需要Node.js环境，可以打包成真正的.ipa（iOS）和.apk（Android）。

### 前置条件
- 安装 [Node.js](https://nodejs.org/)
- 安装 Cordova: `npm install -g cordova`
- iOS打包需要Mac电脑 + Xcode
- Android打包需要Android Studio + JDK

### 打包步骤

#### 1. 创建Cordova项目

```bash
# 进入app-mobile目录
cd app-mobile

# 创建Cordova项目（首次）
cordova create cordova-app com.example.summercheckin 暑假自律打卡
cd cordova-app

# 复制文件到www目录
cp ../index.html www/
cp ../manifest.json www/
cp ../service-worker.js www/
cp ../icon-192.png www/
cp ../icon-512.png www/

# 添加平台
cordova platform add ios
cordova platform add android

# 添加状态栏插件（适配iOS/Android状态栏）
cordova plugin add cordova-plugin-statusbar
```

#### 2. 用提供的config.xml覆盖默认的

将本目录下的 `config.xml` 复制到 `cordova-app/config.xml`

#### 3. 打包iOS

```bash
cordova build ios --release
# 输出在 platforms/ios/build/ 目录
# 用Xcode打开 .xcworkspace 文件，配置签名后Archive导出.ipa
```

#### 4. 打包Android

```bash
cordova build android --release
# 输出在 platforms/android/app/build/outputs/apk/release/ 目录
# 如需发布到应用商店，需要生成签名密钥并重新签名
```

## 方案三：Capacitor打包（推荐，更现代）

Capacitor是Ionic团队推出的替代Cordova的方案，更轻量、性能更好。

### 打包步骤

```bash
# 进入app-mobile目录
cd app-mobile

# 初始化npm项目
npm init -y

# 安装Capacitor
npm install @capacitor/core @capacitor/cli

# 初始化Capacitor
npx cap init 暑假自律打卡 com.example.summercheckin --web-dir .

# 添加平台
npx cap add ios
npx cap add android

# 同步代码
npx cap sync

# 打开Xcode/Android Studio打包
npx cap open ios
npx cap open android
```

---

## 注意事项

1. **图标替换**：当前图标是自动生成的，建议替换为专业设计的图标（1024x1024 PNG）
2. **iOS审核**：如需上架App Store，需准备开发者账号（$99/年）
3. **Android签名**：发布到Google Play需要创建签名密钥
4. **数据持久化**：当前使用localStorage，在App中会被容器隔离，不影响使用

## 推荐的快速上线方案

1. 先用PWA方式部署到服务器（5分钟完成）
2. 用户添加到主屏幕即可使用
3. 如需上架应用商店，再用Capacitor打包
