# 暑假自律打卡 - 亲子版 (PWA App)

## 安装说明

### 方法一：手机安装（推荐）
1. 将本压缩包解压到手机或电脑任意文件夹
2. 在手机浏览器中打开 `index.html` 文件
3. 浏览器会提示"添加到主屏幕"，点击安装即可像原生APP一样使用

### 方法二：浏览器使用
1. 解压本压缩包
2. 用浏览器打开 `index.html` 文件即可使用

### 方法三：部署到服务器
1. 解压后将整个文件夹上传到任意Web服务器或静态托管平台
2. 通过URL访问即可

## 文件说明

- `index.html` — 主应用文件（包含所有功能）
- `manifest.json` — PWA应用配置
- `sw.js` — 离线缓存服务（支持无网络使用）
- `icon-192.png` / `icon-512.png` — 应用图标

## 功能特点

- 支持离线使用（安装后无需网络）
- 可添加到手机主屏幕
- 亲子版打卡功能
- 支持暗黑模式

## 支持平台

- iOS Safari（iOS 11.3+）
- Android Chrome
- 桌面版 Chrome/Edge/Safari
